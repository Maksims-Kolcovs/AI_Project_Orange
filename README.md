**Licence:** Brīvi lietojama akadēmiskiem mērķiem

- Fixed acidity

- Citric acid

- Chlorides

- Quality (kategoriska, vērtības no 3 līdz 8, iespējams grupēt kā *zema*, *vidēja*, *augsta*)
